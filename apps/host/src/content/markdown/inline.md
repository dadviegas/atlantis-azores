# Inline formatting

The basics that every markdown processor supports.

You get **bold**, *italic*, ~~strikethrough~~, and `inline code`.
Links look like [this one to Atlas](https://www.mongodb.com/atlas).

Combine them: **bold with `code` inside**, or *italic with a [link](#)*.
