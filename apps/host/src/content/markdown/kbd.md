# Keyboard

The `<kbd>` HTML tag is rendered through the design system's `<Kbd />`
component, which gives it a pressed-key look.

Press <kbd>Cmd</kbd> + <kbd>K</kbd> to open the command palette.

Common shortcuts:

- <kbd>Cmd</kbd> + <kbd>S</kbd> — save
- <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> — run command
- <kbd>Esc</kbd> — close any panel
- <kbd>?</kbd> — show keyboard help
